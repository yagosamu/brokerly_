# Partners Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver the tenant-isolated Agent and Producer management backend from Brokerly Sprint 16.

**Architecture:** Add a root-level Django app following the existing claims and insurers patterns. Keep tenant enforcement in models, filtered form querysets, defense-in-depth form validation, and `TenantQuerysetMixin` CBVs; expose no brokerage input to clients.

**Tech Stack:** Python 3.13+, Django 6, PostgreSQL, validate-docbr 2.0.0, Docker Compose
